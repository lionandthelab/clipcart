살림해결소 (Clipcart)

Affiliate lifestyle short-form content publishing tool.

Clipcart is an internal tool for a single creator that helps prepare and publish short-form (vertical) lifestyle product-recommendation videos to platforms such as YouTube Shorts, Instagram Reels, and TikTok. Every video is reviewed and explicitly approved by the operator before it is posted. An affiliate disclosure is included in every caption/description.

How it works

  • The operator connects their own platform account via official OAuth.
  • The tool drafts a video, caption, and affiliate disclosure for a product.
  • Nothing is posted automatically — the operator approves each publish action.
  • Publishing uses only official platform APIs.
