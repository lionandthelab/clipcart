# 흥행 홍보숏폼 트렌드 분석 (2026-06-10 리서치)

멀티에이전트 리서치 워크플로우 결과 요약. 영상 엔진이 사용하는 `data/format_profile.json`의 근거 문서.

## 1. 제목 공식 (한국 살림템 쇼츠)

| 패턴 | 예시 |
|---|---|
| 도발형 질문 "아직도 ~하세요?" | 창틀 청소 아직도 물티슈로 하세요? |
| 타깃 지목형 "~인 사람만 보세요" | 싱크대 물 튀는 사람 이거 보세요 |
| 숫자/리스트형 TOP3/BEST10 | 다이소 직원들이 뽑은 화장실 청소템 Top5 |
| 내부자/권위 인용형 | 주방 고수들은 이미 다 쓰는중 |
| 결과/시간 단축 두괄식 | 10초 만에 홀랑 벗겨주는 |
| 원인 궁금증형 "~안 되는 이유" | 냉장고 정리 안 되는 이유 |

- 제목은 100자 한도지만 피드에서 ~40자에서 잘림 → 핵심을 앞 40자에 배치.
- 한국어 기준 20~30자 질문형이 채널 문체 규칙과 가장 잘 맞음.

## 2. 영상 포맷

- **최적 길이**: 25~35초 (제품 홍보형). 60초 초과 시 완주율 급락.
- **장면 구성**: 훅(0-2s, 인트로 금지) → 문제 공감(2-6s) → 제품+가격(6-9s) → 사용 데모(9-19s) → 결과(19-24s) → 단점 1개(24-27s) → CTA+고지(27-30s).
- **자막**: Black/ExtraBold 고딕, 흰색+검정 외곽선, 핵심 단어만 노랑(#FFD400), 경고/가격/단점은 빨강(#FF3B30). 한 줄 6~12자, 화면 세로 중앙~하단 1/3.
- **내레이션**: 해요체 짧은 문장(10~20자), 30초에 7~10문장. TTS 1.1~1.2배속이 실전 표준. 훅 문장만 +15~18%.
- **TTS**: ko-KR-SunHiNeural(1순위), rate +12%, pitch 기본.
- **컷 전환**: 2~3초마다. 마지막 프레임은 결과 장면으로 끝내 루프 재생 유도.
- **음악**: 저작권 리스크 회피를 위해 미사용 (내레이션 중심).

## 3. 썸네일

- Shorts 피드는 자동재생이라 **첫 프레임이 사실상 썸네일** → 훅 장면에 베이크인.
- API `thumbnails.set`은 Shorts에 정상 반영 안 됨(Google Issue #381127084) — 보조용으로만 시도.
- 텍스트 2~7자 초대형, 상단 1/3 배치(피드 하단 잘림 대비), 고대비.

## 4. YouTube Shorts 메타데이터 (2026-06 기준)

- Shorts 분류: 세로/정사각 + 3분 이하 자동 분류. `#shorts` 태그 불필요(무해).
- 해시태그: 노출 증폭이 아닌 주제 분류 신호. 관련성 높은 3~5개.
- categoryId: **26 (Howto & Style)** 권장.
- 유료 광고 포함(paid promotion) 선언: **API 설정 불가** → Studio에서 수동 체크 필요.
- 업로드 quota: 2026-06-01부터 videos.insert 전용 버킷 일 100회 — 하루 1개 업로드는 여유.
- 미검증 OAuth 클라이언트 업로드 시 영상이 비공개로 잠길 수 있음 — 발생 시 OAuth 동의화면 검증 필요.

## 5. 컴플라이언스 (공정위 + 쿠팡 파트너스)

- **확정형 고지 필수** (2024-12 개정): "수수료를 받을 수 있습니다"(조건부) 금지 → **"이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."**
- 동영상: 시작부와 끝부에 표시 + 반복 노출 → 엔진은 전 장면 좌상단 "광고 · 쿠팡 파트너스 수수료 지급" 상시 배너 + 마지막 장면 전문 고지.
- 설명란: 고지를 첫 부분(더보기 접힘 위)에 — 끝부분만 표기는 불인정.
- 한국어 콘텐츠는 한국어 고지 ("AD" 단독 불충분).

## 6. 쿠팡 파트너스 API 운영 메모

- 현행 공식 경로: `/v2/providers/affiliate_open_api/apis/openapi/v1/...`
- 검색 API limit 최대 10, 가격 필터 미지원(클라이언트 필터링).
- 검색 결과 `productUrl`은 이미 추적 링크 — deeplink는 단축링크(coupa.ng / link.coupang.com/a/) 용도.
- **rate limit 주의**: 공식 50회/분이지만 실측 ~10회/시간 보고 다수. 파이프라인은 실행당 검색 5회 제한 + 403/429 시 즉시 중단.
- `subId`는 파트너스에 등록된 채널 ID만 정산 인정 — 미등록 임의값 전달 금지.
- 업로드 최적 시간: 아침 슬롯은 07:00~07:30 KST (출근 피크 선점). 2~3주 후 Studio 시청자 데이터로 보정.

## 7. 성장 후 옵션

- 구독자 1만+ 달성 시 유튜브 쇼핑 제휴(쿠팡 연동, 수수료 ~6.7%)로 영상 내 직접 제품 태그 전환 검토.
